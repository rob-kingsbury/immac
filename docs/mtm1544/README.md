# MTM1544: Web Styles (CSS)

A page's structure is only half the story. This course is about the other half: turning plain, unstyled HTML into a designed, responsive, accessible website using modern CSS. You'll work through layout with Flexbox and Grid, typography and colour, responsive design, and the visual principles that make a page readable, then bring it all together by styling the multi-page site you build in MTM1511.
