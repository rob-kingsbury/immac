# Course Overview

MTM1544 covers styling web pages with modern, hand-coded CSS: the visual and layout layer applied on top of the HTML structure from MTM1511. By the end of the course, you'll have taken a multi-page site from unstyled markup to a responsive, accessible, fully designed website, and be able to explain the decisions behind it.

## Course Learning Requirements

When you have earned credit for this course, you will have demonstrated the ability to:

**CLR 1: Hand code CSS functionality for use in web pages and websites.**
Apply CSS selectors to target elements precisely, explain the cascade, specificity, and inheritance, use CSS properties for typography, colour, spacing, borders, and backgrounds, and maintain a consistent style across pages with external stylesheets.

**CLR 2: Design and lay out web pages for cross-browser and cross-platform application.**
Build responsive layouts with Flexbox and Grid, apply media queries across common breakpoints using a mobile-first approach, test rendering across browsers and platforms, and use browser developer tools to inspect styles and layout.

**CLR 3: Apply accessible and usable visual design principles to styled web pages.**
Meet accessible colour-contrast ratios, use relative units to support text scaling, keep focus states visible and navigation usable without a pointer, and apply visual design principles including hierarchy, contrast, alignment, and proximity.

**CLR 4: Produce fully styled, responsive, multi-page websites that demonstrate technical precision and visual design.**
Apply a coherent design system across pages, integrate CSS into the cross-course HTML project from MTM1511, evaluate design decisions against usability and accessibility standards, and present and explain the choices made.

## How the course is organized

The semester runs in a 7/1/7 pattern (seven teaching weeks, a reading week, then seven more teaching weeks), with weeks 14 and 15 set aside for guided project development and open lab time. Each week here pairs with that same week in MTM1511: you build a page's structure in Web Components, then style it here in Web Styles. See the Weekly Content list in the sidebar for the week-by-week breakdown.

Week 1 finishes the setup started in MTM1511 earlier in the week. You'll confirm your tools work, wire a stylesheet into a page, and publish it live. Real CSS work starts in week 2.
