# Reading Week

No scheduled classes this week. Classes resume the following week with The DOM and CSS Targeting.

The first seven weeks covered a lot of ground, and the second half builds directly on it. If any of the following is still shaky, this is the week to shore it up, because everything from here assumes it.

- **The box model.** Can you predict whether a change needs padding or margin before you try it?
- **Selectors and specificity.** Given two rules that conflict, can you work out which wins without guessing?
- **Flexbox and Grid.** Can you say which one a given layout needs, and why?
- **Mobile-first media queries.** Is your stylesheet written with `min-width` queries building upward?

A useful way to spend an hour or two: open your project, pick one page, and rebuild its layout from scratch without looking at your existing CSS. Whatever you have to look up is what to review.

Otherwise, use the time to catch up on unfinished work and get ahead on your project.
