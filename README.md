# IMM Web Courses

Course content site for Algonquin College's Interactive Media Management
(IMM) program — **MTM1511 (Web Components)** and **MTM1544 (Web Styles)**.

Organized as a set of topic-based lesson modules shared across both
courses, with each course providing its own overview, weekly content, and
resources.

This site is a reference resource only. Assignments and submissions are
handled through Brightspace and student GitHub repos, not here.
